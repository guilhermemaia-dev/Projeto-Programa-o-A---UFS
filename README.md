Introdução:

 O projeto é composto em várias etapas que possuem adições específicas e melhorias constantes com o objetivo de recriar uma aplicação semelhante ao aplicativo distribuído pela Microsoft, Paint. os alunos envolvidos são:

Instalação:
    
	O projeto tem sua última versão estável como o Release mais recente disponível em: https://github.com/guilhermemaia-dev/ Projeto-Programa-o-A---UFS
    
Para instalá-lo em sua máquina siga o passo a passo:
 
 Windows:
     
	 Caso não queira instalar nada adicional, entre no link acima e clique na última release e Source code (zip), extraia o arquivo para o local desejado e execute o seguinte:
         
		 cd "Caminho da pasta que contém o arquivo main"
         python main.py
 
 Linux:
     
	 Caso não queira instalar nada adicional, entre no link acima e clique na última release e Source code (zip), extraia o arquivo para o local desejado e execute o seguinte no seu terminal:
         
		 cd "Caminho da pasta que contém o arquivo main"
         python main.py
 
 MACOS:
     
	 Caso não queira instalar nada adicional, entre no link acima e clique na última release e Source code (zip), extraia o arquivo para o local desejado e execute o seguinte no seu terminal:
        
		 cd "Caminho da pasta que contém o arquivo main"
         python main.py

Remoção:
    
	Basta apagar todos os arquivos da pasta do seu sistema.
